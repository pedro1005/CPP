## CPP Projects

This repository contains various C++ projects that I have worked on, ranging from beginner to advanced level. Each project is designed to demonstrate different C++ concepts and practices.

### Key Concepts Covered:
- Object-Oriented Programming (OOP)
- Memory Management (Pointers, References, Dynamic Allocation)
- Inheritance and Polymorphism
- Function Overloading
- Abstract Classes and Interfaces
- Data Structures and Algorithms

---

## Project List

Below is a list of individual projects and their respective descriptions:

1. **[CPP00: Object-Oriented Programming Basics]**
   - Description: A simple implementation demonstrating the basic principles of OOP, including classes, objects, constructors, destructors, and member functions.
   - Key Concepts: Classes, Objects, Constructors, Destructors

2. **[CPP01: Memory Management]**
   - Description: A project focusing on dynamic memory allocation, pointers, references, and memory leak management.
   - Key Concepts: Pointers, References, Dynamic Memory Allocation

3. **[CPP02: Function Overloading and Polymorphism]**
   - Description: Demonstrates function overloading and basic polymorphism techniques in C++.
   - Key Concepts: Function Overloading, Polymorphism

4. **[CPP03: Inheritance and Virtual Functions]**
   - Description: An implementation of class inheritance and virtual functions to demonstrate runtime polymorphism.
   - Key Concepts: Inheritance, Virtual Functions, Overriding

5. **[CPP04: Abstract Classes and Interfaces]**
   - Description: A project to showcase abstract classes, pure virtual functions, and interfaces in C++.
   - Key Concepts: Abstract Classes, Interfaces, Pure Virtual Functions

---

General Information

    Compiler: These projects are intended to be compiled with g++. Make sure you have the necessary development tools installed.

    Makefile: Each project includes a Makefile to simplify the build process. Use make to compile the project and make clean to clean up the build artifacts.
